## Project Overview
This project analyzes employee attrition at Green Destinations using Python.
It identifies attrition rate and key factors influencing employees leaving the company.

## Objectives
- Calculate attrition rate
- Analyze impact of age, years at company, and income
- Visualize attrition trends

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the program:
   python Green_Destinations_Attrition_Analysis.py

## Output
Graphs are saved automatically in the `outputs` folder.
